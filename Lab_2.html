Course Catalog Description:

This course will cover the creation of web pages and site using HTML, CSS, JavaScript and graphical Application. Networked multimedia distribution technologies are also explored. The design of effective Web site including page layout, user interface design, graphic design, content flow and site structure as well as management of Web site resources including intranet management and design consideration are addressed. Student design and create a major Website with multiple pages and cross-linked structure. Prerequisites:[none] Credit: 2-2-3 Semester Hours.

Course Outcome:

Student completing this course will be able to:

  • Recognize HTML, CSS, JavaScript mark-up and code in web pages/application.
  • Select the proper mark-up tags or code to achieve a particular result.
  • Identify improperly used mark-up and code.
  • Produce modern standards complaint web pages.
  • Deploy web pages to a public server.

Course Objective:

At the conclusion of this course, each successful student will be able to:

  • Explain the client and server architecture of the internet and related web technologies.
  • Use a basic text editor and other software tools to create web pages using HTML, CSS, JavaScript, JQuery nad images.
  • Deploy web pages to a Linux-based web server using ftp.
